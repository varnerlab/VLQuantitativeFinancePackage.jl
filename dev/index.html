VLQuantitativeFinancePackage.jl

The VLQuantitativeFinancePackage.jl package is a Julia package that provides a collection of functions and types useful for quantitative finance. The package is designed to be simple and easy to use, and it is suitable for students, researchers, and practitioners in the area of quantitative finance.

Installation

The package can be installed by running the following command in the Julia REPL:

using Pkg
Pkg.add(url="https://github.com/varnerlab/VLQuantitativeFinancePackage.jl.git")
